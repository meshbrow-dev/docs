# Get Profile

Retrieve a profile's configuration, cookies, and usage statistics.

<ParamField path="id" type="string" required>
  The profile ID.
</ParamField>

## Request

```bash
curl https://api.meshbrow.dev/v1/profiles/prof_x9y8z7w6 \
  -H "Authorization: Bearer mb_live_..."
```
