# Fleet Status

Retrieve the current status of a fleet and all its sessions.

<ParamField path="id" type="string" required>
  The fleet ID.
</ParamField>

## Request

```bash
curl https://api.meshbrow.dev/v1/fleet/flt_m4n5o6p7 \
  -H "Authorization: Bearer mb_live_..."
```
